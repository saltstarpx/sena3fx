# 要件整理：本多廣大トレードPDCAエージェント

## ユーザーの目標
- 2026年年間利益1,000万円（現在進捗：約130万円）
- OANDAゴールドCFD（XAUUSD）・シルバーCFD（XAGUSD）の裁量トレード
- PDCAサイクルを完全自動化したい（手動ゼロ）

## トレード手法：やがみメソッド
### エントリー5条件
1. レジサポ位置（日足/4時間足レベル）
2. ローソク足の強弱
3. プライスアクション（ピンバー・包み足・ヒゲ消費等）
4. チャートパターン（三角保ち合い・カップハンドル・ダブルボトム等）
5. 足更新タイミング（日足/4時間足の更新タイミング）

### 評価基準
- 4-5条件 → A評価（高品質）
- 3条件 → B評価（許容）
- 2条件以下 → C評価（養分エントリー）

## 養分行動チェック
- 祈りホールド（損切り不履行）
- 計画外ナンピン
- 根拠2条件以下でのエントリー
- 安易な建値ストップ
- アジア時間（JST 6:00-16:00）のブレイクアウト
- 大陽線・大陰線への逆張り
- トレンドレス相場への突入

## 蓄積済み知見
### 強み（継続）
- XAUUSD 8-24時間保有：勝率高・RR優秀 → メイン戦略
- XAGUSD スイング24時間超：勝率100% → 継続
- ロンドン/NYブレイクアウト：フォロースルー良好

### 弱み（廃止・改善）
- XAGUSD スキャルピング1時間未満：勝率27%・赤字 → 完全廃止
- アジア時間ブレイクアウト：ストップ狩られやすい
- ロンドン時間XAUUSD：RR低下傾向 → ロット抑制

### 検証すべき仮説
1. NYセッション（22-6JST）は8-24時間保有との相性が特に良いか
2. 週明け月曜は週次トレンド方向にバイアスがあるか
3. 指標発表後のXAUUSDはどのセッションで最もRRが取れるか

## バックテスト合格基準（SECTION 5）
- プロフィットファクター：1.5以上
- 最大ドローダウン：資金の10%以下
- 勝率：50%以上（RR2.0以上の場合）
- サンプル数：30トレード以上

## 資金管理ルール
- 1トレード最大リスク：総資金の2%
- 複利運用法（デイトレ〜スイング軸）
- 損切り位置：最後の押し安値/戻り高値
- 建値ストップ禁止
- ロット計算：(総資金 × 0.02) ÷ (損切り幅 × 1ロットあたりの価値)

## 定量分析項目
1. 通貨ペア別（XAUUSD/XAGUSD）の勝率・RR・損益
2. 保有時間別パフォーマンス（<1h / 1-8h / 8-24h / 24h超）
3. セッション別（アジア6-16JST / ロンドン16-22JST / NY22-6JST）
4. 曜日別パフォーマンス（月〜金）
5. エントリー方向別（ロング/ショート）
6. 月次・週次損益推移
7. 最大連続勝ち・負け数
8. プロフィットファクター（総利益÷総損失）
9. バルサラ破産確率チェック
