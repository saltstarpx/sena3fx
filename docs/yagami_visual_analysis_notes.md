# やがみ式プライスアクション：画像解析による視覚的特徴のまとめ

## 1. ローソク足の強弱と意味（視覚的定義）
*   **大陽線・大陰線**: 逆張り厳禁。トレンドの継続を示す。
*   **下髭陽線（安値圏）**: 
    *   前の陰線が大きければ大きいほど、反発の期待値が高い（輪ゴムの原理）。
    *   実体部分も長い方が強い。
*   **ピンバー**:
    *   下髭は長ければ長いほど安心感がある。
    *   単体ではなく、更新された次の足が陽線なら反転気配、陰線ならレンジ継続。
*   **リバーサルロー（最強のシグナル）**:
    *   安値圏で確定した際に、ほぼ底値が確定する形状。
    *   上位足（4h, 日足）でこの形が出ると非常に強力。

## 2. エントリーのタイミングと場所
*   **「どこで止まるか」ではなく「どこで止まったか」**:
    *   急落の最中に買うのではなく、一度止まってから水平線を引いてエントリーを検討する。
    *   最低でも15分足で1本は逆の色（陽線）が付いてから。
*   **二番底・二番天井の重視**:
    *   1回目のアタック（左肩）ではなく、2回目のアタック（右肩）での反転を確認する。
    *   横軸（時間）を待つことで、連続損切りを避ける。

## 3. 実体の揃いとインサイドバー（核心ロジック）
*   **実体が揃っている場所**:
    *   レンジ内で複数のローソク足の実体（終値・始値）が水平に並んでいる箇所。
    *   ある程度の「実体の厚み」が必要。薄すぎる場合は信頼度が低い。
    *   この実体の厚み＝リバの強さと判断する。
*   **インサイドバーの活用**:
    *   前回のローソク足の高値安値を更新せずに確定する足。
    *   これが連続することで「逆三尊」や「ダブルボトム」の右肩が形成される。
*   **逆三尊の先回り**:
    *   「逆三尊になってから」ではなく、「ここでリバったら逆三尊になる位置」でポジションを取る。
    *   具体的には、実体が揃い、インサイドバーが出現し始めた右肩の部分。

## 4. 決済とリスク管理
*   **上髭埋めムーブ**:
    *   スパイクハイの髭を実体で埋めながら高値を更新する動き。
    *   実体のレジスタンスで部分利確、髭埋め完了で全利確。
*   **背（ストップ）の置き方**:
    *   直近の安値（髭先）を明確な「背」として設定する。
    *   背が明確でない場所ではエントリーしない。

## 5. バックテスト（RUN-009）への反映方針
*   **実体の揃い判定**: 単なる価格比較ではなく、一定期間内の終値の分散が小さい箇所を特定するロジックを導入。
*   **インサイドバー・フィルタ**: エントリー条件に「前足のレンジ内での確定」を追加。
*   **足確定直前のエントリー**: 15分足の14分時点など、確定間際での判定をシミュレートする。
