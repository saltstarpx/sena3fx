ボラティリティフィルターの分析と影響

フィルター有効時 (デフォルト)

シグナル発生数: 0件

ATRが閾値（スプレッドの3倍 = 0.006）を常に下回っていたため、すべてのエントリーがブロックされました。2月の相場環境では、この条件は厳しすぎることが判明しました。

フィルター無効化時 (デバッグ)

シグナル発生数: 115件

フィルターを外すと大量のシグナルが発生しましたが、PF 0.046という極めて低い結果に。ボラティリティが低い局面での「髭」狙いは、ダマシに遭う確率が非常に高いことが実証されました。

教訓

フィルター自体は「無駄な負け」を減らすために不可欠です。しかし、2月の低ボラティリティ環境に合わせるには、閾値をスプレッドの0.5〜1.0倍程度まで緩和する必要があります。

結論：フィルターは機能しているが、閾値の最適化が必要。低ボラティリティ下でのロジックの脆弱性が浮き彫りになった。
