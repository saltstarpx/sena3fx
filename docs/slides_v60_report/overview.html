Yagami MTF v60 戦略概要

マルチタイムフレーム分析

15分足、1時間足、4時間足の3つの時間軸を監視し、相場の全体像を把握します。

トレンドフィルター

1時間足と4時間足のEMA（20/75）を用いて強力なトレンド（壁）を特定し、その方向にのみエントリーします。

エントリートリガー

15分足のローソク足の「髭」の長さをATR（平均真の範囲）と比較し、反転の可能性が高いポイントを狙います。

エントリータイミング

1分足まで時間軸を落とし、15分足で特定した方向への押し目や戻りを精密に捉えてエントリーします。

リスク管理

損切りはATRの3倍に設定し、利益確定はリスクリワード比5.0で固定。明確なルールで資金を保護します。

動的タイムディケイ

相場のボラティリティ（ATR）に応じてポジションの最大保有時間を調整し、効率的な取引を目指します。
