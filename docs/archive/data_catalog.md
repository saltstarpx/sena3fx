# データカタログ

本ドキュメントは、本プロジェクトで使用されるデータの一覧、期間、および出所を記録します。

## 1. ティックデータ

- **出所**: Polygon.io
- **期間**: 未定（バックテストの期間による）
- **内容**: 各銘柄のBid/Ask価格、タイムスタンプ
- **備考**: 生データであり、OHLCバー生成の元データとなる。

## 2. OHLCバーデータ

- **出所**: ティックデータから生成
- **期間**: 未定
- **内容**: 始値、高値、安値、終値、スプレッド、ティックカウント
- **備考**: バックテストエンジンへの主要入力データ。

## 3. バックテスト結果データ

- **出所**: バックテストエンジン (`lib/backtest.py`) から出力
- **期間**: バックテスト実行期間
- **内容**: 損益、勝率、最大ドローダウン、プロフィットファクターなどの統計情報、個別の取引履歴
- **備考**: `results/performance_log.csv` および `trade_logs/broker_history_*.csv` に記録される。
