# やがみ式 Note画像再解析レポート (v4)

RUN-011（1月データ）での「スプレッド即死」という課題を踏まえ、画像を再解析した結果、以下の「視覚的な核心」を再定義しました。

## 1. 「レジサポ」の真の定義 (Page 2-3)
*   **実体と髭の間**: 水平線を引くべき場所は「髭先」だけでなく、**「実体（終値/始値）」と「髭」の間**にある。
*   **強固な壁**: 単発の足ではなく、複数の足の実体が水平に揃っている箇所が「強固なレジサポ」となる。
*   **時間軸の倍率**: 15分足でエントリーする場合、その12倍〜16倍の時間軸（4時間足）のレジサポがメインの戦場となる。

## 2. 「ミスプライス（髭）」と「適正価格（実体）」 (Page 4-5)
*   **ミスプライスの埋め**: 髭を付けた後、その髭を実体で埋めてくる動き（髭埋め）は、反転の失敗と継続のサイン。
*   **実体ベースの安値更新**: 髭先（最安値）を更新していなくても、**実体ベースで安値が切り下がっている**場合、その価格帯でのサポート力は消失している。
*   **戦略への反映**: 1分足の髭先を背にする際、その「髭」が上位足（15分・4H）において「適正価格の外側（ミスプライス）」にあることを確認する必要がある。

## 3. 「スプレッド即死」への対策
*   **ノイズの許容**: 1分足の髭先ジャストにSLを置くのではなく、**「実体が揃っているライン（適正価格の境界）」**を背にし、その外側にスプレッド＋αのバッファを持たせる必要がある。
*   **横軸の強度**: 1分足での「止まった」判定を、単なる価格の近似ではなく、**「実体がスプレッド以上の幅を持ってレンジを形成しているか」**で判定する。

## 4. 15分足の強弱パターン (Page 6)
*   反転を狙う際、単に安値圏にいるだけでなく、**「下髭の長い大陽線」や「前の陰線を実体で包む足（包み足）」**などの、視覚的に強い買いのサインを15分足で待つ必要がある。
