# Claude Code への実装指示書：やがみ式3層MTF戦略 (v3.0)

本ドキュメントは、Manus による「やがみ式プライスアクション」の検証結果（RUN-010）に基づき、実戦用トレードエンジンの実装を指示するものです。

## 1. 検証結果の概要 (RUN-010)

やがみ氏本人から得られた「3層構造」を実装した結果、劇的なパフォーマンス向上が確認されました。

*   **検証データ**: USD/JPY 1分足 (2026-03-04 ~ 2026-03-05)
*   **プロフィットファクター (PF)**: **3.137**
*   **勝率 (WR)**: 10.0% (低勝率・高RR型)
*   **主要な改善点**: 1分足の髭先を「背」にした極限のタイトSLと、4H壁を目標とした高RRトレード。

## 2. 実装すべき3層ロジックの詳細

以下の3つの時間軸を同期させ、条件が重なった瞬間のみ執行するロジックを構築してください。

### ① 4時間足 (H4): 環境認識と「場所」の特定
*   **トレンド判定**: 直近20本の高値・安値の切り上げ/切り下げで判定。
*   **主要な壁 (Wall)**: 直近20本の最高値（Wall High）と最安値（Wall Low）を特定。
*   **役割**: エントリーの方向性（Long/Short/Neutral）を限定し、最終的な利確目標（TP）を定義する。

### ② 15分足 (M15): パターン認識と「タイミング」の探索
*   **実体の揃い (Body Alignment)**: 直近3本の実体（終値）の標準偏差がATRの0.20倍以下であることを「壁の形成」と定義。
*   **インサイドバー**: 前足のレンジ内に収まる足の出現を「エネルギーの蓄積」とみなす。
*   **エントリーゾーン**: 4Hの壁から 2.0 * H4_ATR 以内の領域のみを有効とする。

### ③ 1分足 (M1): 精密な執行と「背」の確定
*   **ボラティリティ収束**: 1分足のレンジが直近5本の平均レンジの0.6倍以下に収束（横軸の形成）していること。
*   **放れの確認**: 収束後の最初の陽線（Long）または陰線（Short）の確定でエントリー。
*   **極限の損切り (Tight SL)**: **エントリーした1分足の安値（Long）または高値（Short）を「背」とする。** (バッファは最小限: 0.005程度)

## 3. 実装上の技術的要件

*   **マルチタイムフレームの同期**: 1分足のティック受信ごとに、15分足と4時間足の状態を正しく更新・参照すること。
*   **リスク管理**: 
    *   1トレードのリスクを資金の 2% に固定。
    *   SLが極めてタイトなため、スリッページを考慮したポジションサイズ計算を行うこと。
*   **利確 (TP) 設定**:
    *   4時間足の反対側の壁を基本目標とする。
    *   最低でもリスクリワード比 (RR) 1.5 以上を確保できない場合はエントリーを見送る。
