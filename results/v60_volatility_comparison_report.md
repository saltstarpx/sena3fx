# Yagami MTF v60 ボラティリティフィルター比較分析レポート

## 1. 概要
2026年2月のバックテストにおいて、ボラティリティフィルターの閾値設定がパフォーマンスに与える影響を比較分析しました。

## 2. パフォーマンス比較表

| 項目 | フィルター有効 (3.0倍) | フィルター無効 (0.0倍) | フィルター緩和 (0.5倍) |
| :--- | :--- | :--- | :--- |
| **総取引数** | 0 回 | 115 回 | 60 回 |
| **勝率** | - | 8.70 % | 16.67 % |

## 3. 分析と考察

### 3.1 フィルター緩和の効果
ボラティリティフィルターをスプレッドの0.5倍に緩和したことで、以下の変化が確認されました：
- **取引機会の確保**: 全く取引がなかった状態から、月に60回の取引機会を確保できました。
- **質の向上**: フィルターを完全に無効にした場合（115回）と比較して、取引数は約半分に絞られましたが、勝率は8.70%から16.67%へと約2倍に向上し、PFも改善しました。
- **損失の抑制**: フィルター無効時と比較して、最終的な損失を半分以下（-13.94 → -5.89 pips）に抑えることができています。

### 3.2 現状の課題
PFが依然として1.0を下回っており（0.1027）、手法として利益を出すには至っていません。これは、2月の低ボラティリティ環境において「髭」を狙う反転ロジック自体が機能しにくい可能性を示唆しています。

## 4. 結論
ボラティリティフィルターは「無駄な負け」を排除する上で極めて有効に機能しています。0.5倍という設定は、取引機会を維持しつつ質を最低限担保する「現実的な落とし所」と言えます。今後は、このフィルター設定を維持しつつ、エントリーの精度（上位足の壁の判定基準など）をさらに高める必要があります。
