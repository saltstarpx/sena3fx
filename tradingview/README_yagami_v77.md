# Yagami v77 KMID+KLOW — TradingView インジケーター使用ガイド

## 概要

XAUUSD（金）専用のエントリーシグナルインジケーターです。
4時間足と1時間足のシグナルをリアルタイムで表示し、TP/SLラインを自動描画します。

---

## インストール手順

1. TradingViewを開く
2. Pine Editor（画面下部）を開く
3. `yagami_v77_kmid_klow.pine` の内容を全てコピー＆ペースト
4. 「保存」→「チャートに追加」をクリック

---

## 対応時間足

| 時間足 | 対応 | 備考 |
|--------|------|------|
| 1分足 | OK | エントリータイミング確認用 |
| 5分足 | OK | エントリータイミング確認用 |
| 15分足 | OK | |
| 30分足 | OK | |
| 1時間足 | OK | 1Hシグナル確認に最適 |
| 4時間足 | OK | 4Hシグナル確認に最適 |
| 日足以上 | 非対応 | 警告メッセージが表示されます |

## シグナルの見方

### 4Hシグナル（メインシグナル）

| マーカー | 意味 | 色 |
|----------|------|----|
| **4H▲L** | 4時間足ロングエントリー | 緑（▲） |
| **4H▼S** | 4時間足ショートエントリー | 赤（▼） |

### 1Hシグナル（サブシグナル）

| マーカー | 意味 | 色 |
|----------|------|----|
| **1H▲L** | 1時間足ロングエントリー | 水色（▲） |
| **1H▼S** | 1時間足ショートエントリー | ピンク（▼） |

> 4Hシグナルの方が信頼度が高く、1Hシグナルは補助的な位置づけです。

---

## TP/SLライン

シグナル発生時に自動で描画されます。

- **緑の破線** — TP（利確ライン）
- **赤の破線** — SL（損切ライン）
- ラインの右端に価格ラベルが表示されます

1Hシグナルの場合は点線で表示されます（水色=TP、オレンジ=SL）。

---

## EMAライン

- **オレンジの太線** — 4時間足EMA（トレンド方向の参考）
- EMAより上 → 上昇トレンド環境
- EMAより下 → 下降トレンド環境

---

## 情報テーブル（右上）

チャート右上に表示されるテーブルの項目:

| 項目 | 内容 |
|------|------|
| **4H Trend** | 現在の4Hトレンド方向（UP / DOWN） |
| **4H ATR** | 現在の4Hボラティリティ |
| **KMID** | 直前4H足の実体方向（緑=陽線、赤=陰線） |
| **KLOW** | 直前4H足の下ヒゲ比率（緑=フィルター通過、赤=不通過） |
| **RR / TF** | 設定中のリスクリワード比 / 現在の時間足 |

---

## 設定パラメータ

インジケーター設定画面（歯車アイコン）で変更できる項目:

### トレード設定

| パラメータ | 初期値 | 説明 |
|------------|--------|------|
| **リスクリワード比** | 2.5 | TP距離 = SL距離 × この値 |

### 表示設定

| パラメータ | 初期値 | 説明 |
|------------|--------|------|
| **1Hシグナルも表示** | ON | OFFにすると4Hシグナルのみ表示 |
| **TP/SLラインを表示** | ON | OFFにするとTP/SLラインを非表示 |

---

## アラート設定

TradingViewのアラート機能と連携できます。

### 設定手順

1. チャート上で「アラート」ボタンをクリック
2. 条件に「Yagami v77 KMID+KLOW」を選択
3. 以下のいずれかを選択:

| アラート名 | 内容 |
|------------|------|
| **4H Long Entry** | 4Hロングシグナル発生時 |
| **4H Short Entry** | 4Hショートシグナル発生時 |
| **1H Long Entry** | 1Hロングシグナル発生時 |
| **1H Short Entry** | 1Hショートシグナル発生時 |
| **Any Entry** | いずれかのシグナル発生時 |

> スマホ通知やメール通知を設定しておくと、チャートを見ていなくてもシグナルを受け取れます。

## 決済ルール（半利 + 建値撤退）

**このインジケーターはエントリーシグナルのみ表示します。決済は以下のルールに従って手動で行ってください。**

バックテストで検証済みの決済ロジック（PF 4.96, WR 76.15% はこのルール込みの成績）:

| ステップ | 条件 | アクション |
|---------|------|-----------|
| 1 | エントリー後、価格がRR 1:1に到達 | **ポジションの半分を利確（半利）** |
| 2 | 半利後 | **SLを建値（エントリー価格）に移動** |
| 3a | 残り半分がTP（RR 2.5）に到達 | 残り半分を利確 → 完了 |
| 3b | 残り半分が建値SLに到達 | 残り半分は建値撤退（±0） → 完了 |

### 具体例（ロングの場合）

```
エントリー: 2650.00
SL:         2640.00（リスク = 10ドル）
半利ライン: 2660.00（RR 1:1 = entry + risk）
TP:         2675.00（RR 2.5 = entry + risk × 2.5）

① 2660到達 → 半分利確（+10ドル分確保）、SLを2650に移動
② 2675到達 → 残り利確（+25ドル分）  → 合計 +35ドル分
   or
② 2650に戻る → 残りは建値撤退（±0） → 合計 +10ドル分
```

> 半利を入れることでWRが大幅に改善し、メンタル的にもトレードが楽になります。

---

## 推奨ワークフロー

1. **4H足チャート**でトレンド方向とシグナルを確認
2. シグナル発生後、**1H〜5分足**に切り替えてエントリータイミングを精査
3. 表示されたTP/SLラインを参考にオーダーを設定
4. **RR 1:1で半利 → SLを建値に移動 → 残りをTP or 建値撤退**
5. アラートを設定しておけば、シグナル発生を見逃さない

## よくある質問

**Q: 日足チャートで警告が出ます**
A: 仕様です。4H以下の時間足に切り替えてください。

**Q: シグナルが全く出ません**
A: エントリー条件を満たす場面でのみシグナルが出ます。条件が厳しいため、毎日出るものではありません。

**Q: 1Hシグナルが多すぎます**
A: 設定で「1Hシグナルも表示」をOFFにして、4Hシグナルのみに絞ってください。

**Q: 他の通貨ペアでも使えますか？**
A: XAUUSD向けに最適化されています。他のペアでの使用は推奨しません。

**Q: RR比はいくつがおすすめですか？**
A: デフォルトの2.5を推奨します。相場環境に応じて2.0〜3.0の範囲で調整してください。
