# Yagami v78 — 5条件スコアリング インジケーター 取扱説明書

## 概要

やがみメソッドの5条件をTradingView上でリアルタイム評価し、エントリーシグナルとTP/SLを自動表示するインジケーターです。

**対象**: XAUUSD（金）
**推奨時間足**: 15分〜4時間足（4H以下で動作）
**ベース**: Python `lib/yagami.py` の5条件判定ロジックを Pine Script v6 に完全移植

---

## インストール方法

1. TradingViewを開く
2. チャート画面で **Pine Editor** を開く（画面下部のタブ）
3. `yagami_v78_5conditions.pine` の全コードをコピー＆ペースト
4. **「インジケーターを追加」** をクリック
5. チャート上に「Yagami v78 5Cond」が表示される

---

## 5条件の説明

シグナルは以下5つの条件を総合評価して発生します。

| 条件 | 内容 | 詳細 |
|------|------|------|
| **C1: レジサポの位置** | 過去10本のS/R水平線に価格が近いか | クラスタリング方式で自動検出 |
| **C2: ローソク足の強弱** | 大陽線・包み足・ハンマー・ピンバー等 | プルバック（弱い逆方向足）も検出 |
| **C3: プライスアクション** | リバーサル・ダブルボトム/トップ・ヒゲ埋め・実体揃い | 過去10本から検出 |
| **C4: チャートパターン** | 逆三尊/三尊・フラッグ・三角持ち合いブレイク | スイングポイントベースで検出 |
| **C5: 足更新タイミング** | 上位足の更新タイミング + 日足方向 | 上位足の色（陽線/陰線）で方向加点 |

### 評価グレード

| グレード | スコア | 意味 |
|----------|--------|------|
| **A** | 4〜5 | 高品質エントリー。自信を持ってポジション取れる |
| **B** | 3 | 慎重エントリー。ロットを抑えて入る |
| **C** | 2以下 | エントリー禁止（養分になる） |

## シグナルの見方

### チャート上の表示

| 表示 | 意味 |
|------|------|
| **4H▲A** (緑・大) | 4H足ロング・A評価 |
| **4H▲B** (緑・小) | 4H足ロング・B評価 |
| **4H▼A** (赤・大) | 4H足ショート・A評価 |
| **4H▼B** (赤・小) | 4H足ショート・B評価 |
| **1H▲** (水色・極小) | 1H足ロング |
| **1H▼** (ピンク・極小) | 1H足ショート |

### スコア内訳ラベル

シグナル発生時、下記形式で内訳が表示されます（「スコア内訳を表示」ON時）:

```
A4 [S●●●●]
```

- 1文字目: グレード（A/B）
- 2文字目: スコア（3〜5）
- 角括弧内: 5条件の成否
  - 位置1 (C1): `S`=サポート付近, `R`=レジスタンス付近, `-`=なし
  - 位置2 (C2): `●`=ローソク足パターン検出, `-`=なし
  - 位置3 (C3): `●`=プライスアクション検出, `-`=なし
  - 位置4 (C4): `●`=チャートパターン検出, `-`=なし
  - 位置5 (C5): `●`=足更新タイミング, `-`=なし

### TP/SLライン

- **緑の破線**: TP（利確ライン）
- **赤の破線**: SL（損切りライン）
- SLはスイングロー/ハイから自動算出（ATR下限保証付き）
- TPはSLからの距離 × リスクリワード比で算出

### 情報テーブル（右上）

| 項目 | 内容 |
|------|------|
| 4H Trend | EMA50に対するトレンド方向（UP/DOWN） |
| 4H ATR | 現在の4H ATR値 |
| Min Score | 設定中の最低スコア |
| RR | リスクリワード比 |
| TF | 現在の表示時間足 |

## 設定項目

設定画面（歯車アイコン）で調整できるパラメータ:

### トレード設定

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| **リスクリワード比** | 2.0 | TP計算に使用。2.0 = リスクの2倍を狙う |
| **最低スコア** | 3 | 3=B評価以上でシグナル、4=A評価のみ |

### 表示設定

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| **1Hシグナルも表示** | OFF | ONにすると1H足のシグナルも表示 |
| **TP/SLラインを表示** | ON | TP/SLの破線ラインを表示 |
| **スコア内訳を表示** | ON | 5条件の内訳ラベルを表示 |

---

## 推奨設定

### 初心者向け（安全重視）
- 最低スコア: **4**（A評価のみ）
- RR比: **2.0**
- 1Hシグナル: **OFF**

### 中級者向け（バランス型）
- 最低スコア: **3**（B評価以上）
- RR比: **2.0**
- 1Hシグナル: **ON**（4Hと同方向の確認用）

---

## 内蔵フィルター（自動適用）

以下のフィルターが自動的に適用され、低品質シグナルを除外します:

| フィルター | 条件 | やがみの教え |
|-----------|------|-------------|
| **大陽線/大陰線 逆張り禁止** | 直前が大陽線ならロング禁止、大陰線ならショート禁止 | 「逆らうと死にます」 |
| **トレンドレス検出** | Doji 4本連続でシグナル停止 | レンジではノーポジ |
| **アジア時間ブレイクアウト除外** | UTC 21:00〜07:00の三角持ち合いブレイクを無視 | 「アジア時間のブレイクアウトは危険」 |
| **リスク上限** | SLが ATR×3.0 を超えるシグナルを除外 | リスク過大を防止 |
| **リスク下限** | SLが ATR×0.3 未満のシグナルを除外 | 薄利すぎを防止 |

## アラート設定

TradingViewのアラート機能と連携できます。

1. チャート上で **アラート作成**（時計アイコン）
2. 条件で「Yagami v78 5Cond」を選択
3. 以下から選択:
   - **4H Long Entry**: 4Hロングシグナル
   - **4H Short Entry**: 4Hショートシグナル
   - **1H Long Entry**: 1Hロングシグナル
   - **1H Short Entry**: 1Hショートシグナル
   - **Any Entry**: 全シグナル

---

## 使い方の流れ

1. XAUUSDのチャートを開き、**15分足〜4時間足**に設定
2. インジケーターを追加
3. シグナル（▲▼）が出たら、**スコア内訳**を確認
4. **TP/SLライン**を参考にエントリー
5. 必要に応じてアラートを設定

### エントリー判断のポイント

- **A評価（スコア4-5）**: 自信を持ってエントリー
- **B評価（スコア3）**: ロットを控えめに。他の根拠も合わせて判断
- **4Hと1Hが同方向**: より信頼度が高い
- **TP/SLラインが表示されない**: リスクが大きすぎ or 小さすぎ → 見送り

---

## 注意事項

- このインジケーターはシグナル補助ツールです。最終判断は自己責任で行ってください
- 4H足以上の時間足では正しく動作しません（警告メッセージが表示されます）
- 1Hシグナルは4Hトレンド方向と一致するものだけ表示されます
- EMA50ラインがオレンジ色で表示されます（4Hトレンドの参考用）
